# Code Samples

This folder contains code examples referenced in the blog post.

## Files

- `example.js` - JavaScript implementation of markdown utilities

## Running the Code

### Prerequisites
- Node.js 14+ installed

### Execution

```bash
node example.js
```

Or import in another file:

```javascript
const { createMarkdownExample } = require('./example.js');
```

## Notes

All code samples are meant to be educational examples and can be freely used and modified.
